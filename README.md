# javiergbas.com

My personal website. A space to share my work, experiments, and things I find interesting — built and iterated in public.

## Stack

- [Vite](https://vite.dev/) — build tool and dev server with HMR
- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/) — via `@tailwindcss/vite` plugin
- [Motion](https://motion.dev/) — animations and scroll effects
- [Lucide React](https://lucide.dev/) — icons

## Getting started

```bash
# Install dependencies
npm install

# Start the dev server with hot reload
npm run dev
```

The site will be available at `http://localhost:5173`.

## Other commands

```bash
# Type-check and build for production
npm run build

# Preview the production build locally
npm run preview

# Lint
npm run lint
```
